Это — инструкция для использования сих письмен для получения доступа в http://www.humansnotinvited.com/.
Эта программа была писана на Windows 10, потому, возможно, на некоторых других компьютерах это не будет работать.

1. Скачать эту всю группу файлов на ПК и установить туда Python 3.6 или выше.
2. Открыть `action.py`, скопировать полный текст кода страницы с http://www.humansnotinvited.com/. Я делал это так:
  Открытие кода страницы — `Ctrl + U`.
  Скопировать весь текст со страницы — `Ctrl + A` (для выбора) + `Ctrl + C` (для копирования).

3. Вставить скопированный текст в консоль и на новой строке ввести стоп-слово (по умолчанию — "STOP") на новой строке [и нажать `Enter`]. По идее, тогда ответ вскоре будет выведен на экран. P.S. У меня на ПК это длилось около 10 сек.
